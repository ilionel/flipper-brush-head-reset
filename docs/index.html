🪥 Brush Head Reset

A Flipper Zero app to read and reset the wear counter stored on the NTAG213 chip inside many smart toothbrush heads — so you can keep using a head you judge still good, instead of binning it on a fixed timer. For heads you own.

⬇ Download .fap Source on GitHub Tag memory map

What it does

Read

Model name, usage gauge (% and minutes) and the per-head password — computed from the tag, never guessed.

Reset to new

Sets the brushing counter back to 0%. Set any usage value, or save & restore a head's state.

Head fields

Decodes the on-tag data: model/type, rated life (months), last intensity & brush mode, security config.

Lock probe

Non-destructive map of which pages are writable vs factory-locked — write-back-identical, nothing is altered.

Install

  1. Open the latest release and download the .fap matching your firmware:
    brush_reset-official-release.fap, …-official-dev.fap, …-unleashed.fap or …-momentum.fap.
  2. Connect your Flipper and open qFlipper (or the SD card directly).
  3. Copy the .fap into SD Card / apps / NFC.
  4. On the Flipper: Apps → NFC → Brush Head Reset.
  5. Pop the head off the handle, lay its base flat on the back of the Flipper, hold still, and pick an action.

The head's coil is tiny — stable coupling matters, especially for a write. Transient drops are retried automatically; a write is lockout-safe (the password is computed and sent once, so the tag's 3-strikes permanent lock is never reachable).

How it works

Each head carries an NTAG213 NFC tag. One page holds the accumulated brushing time in seconds; the handle reads it to decide when to nag for a replacement. With the head's own per-head password (derived from its UID + factory code) the app writes that page back to a value of your choice. The full reverse-engineered layout — every page, field encoding, lock bit and the password algorithm — is in the tag memory map.
